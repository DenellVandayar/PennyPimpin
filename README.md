# 📱 Project Description

**Penny Pimpin'** is a mobile budgeting application designed to help users track, categorise, and manage their expenses efficiently.

The app allows users to:
- Track spending habits
- Create expense categories
- Set monthly budget goals
- Analyse expenditure using graphs and progress bars
- Search expenses over custom time periods

The application focuses on simplicity and ease of use through an interactive and visually appealing user interface.

---

# 🚀 Features

- Add, edit, and delete expense categories
- Set monthly budget goals with minimum and maximum limits
- Assign limits to individual categories
- View expenditure for each category
- Search expenses over a selected period
- View graph analytics for monthly expenditure
- RoomDB integration for persistent local storage
- Firebase integration for online data storage *(partially implemented due to time constraints)*

# 📖 How to Run the Project Locally

1. Clone the repository

```bash
git clone https://github.com/YourUsername/PennyPimpin.git
```

2. Open the project in Android Studio

3. Sync Gradle files and build the project

4. Run the application on:
- An Android emulator
- A physical Android device
